How to Buy AZC

  1. Install MetaMask and connect to BNB Smart Chain (BEP-20).
  2. Fund your wallet with BNB for swaps and gas fees.
  3. Open PancakeSwap V3 and connect wallet, or use this swap link: https://pancakeswap.finance/swap?outputCurrency=0xB9697BFC6eFA3e4893d0B1Ee75838eBE35AcedAd
  4. Enter BNB or AZC amount, set slippage (start with 1–2%), approve token if prompted, then confirm the swap.
  5. After swap, add token to MetaMask using the contract address: 0xB9697BFC6eFA3e4893d0B1Ee75838eBE35AcedAd
